# Build Eseguibile Backend

Questa cartella contiene tutti i file necessari per creare l'eseguibile del backend.

## Quick Start

**Windows:** Doppio click su `build.bat`

**Linux/Mac o da terminale:**
```bash
python build_exe.py
```

## File Contenuti

- `build_exe.py` - Script Python per la build
- `JumpAnalyzerBackend.spec` - File di configurazione PyInstaller
- `build.bat` - Script batch per Windows
- `BUILD_README.md` - Documentazione completa
